# Mecris Tools Directory

This directory contains specialized auxiliary tools and research components for the Mecris ecosystem.

## 📂 Contents

- **`talktype/`**: High-performance local voice-to-text service using OpenAI Whisper.
- **`echokit/`**: Research tool for low-latency audio feedback and processing.

## 🛠️ Environment Management

All tools in this directory use isolated virtual environments managed by `uv`. 

**Critical Mandate**: Never share virtual environments between tools.

👉 **[Read the Environment Management Guide](ENV_MANAGEMENT.md)** for detailed setup and build instructions.
